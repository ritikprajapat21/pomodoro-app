# Pomodoro App: React, Redux, Typescript

A productivity app designed to increase your productivity using pomodoro technique.

# Tech Stack Used

1. React
2. Redux

# Local Development

- Clone this project

```
git clone https://github.com/ritikprajapat21/pomodoro-app.git
```

- Change directory

```
cd pomodoro-app
```

- Install dependency

```
npm install
```

- Run the app

```
npm run dev
```
